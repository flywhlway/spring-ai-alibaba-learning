<threat_model>
## Trust Boundaries
- 用户查询 → FlowAgent / Supervisor ReactAgent → 子 Agent tools → DashScope
- 43 Client(18143) → Nacos discovery → A2A Server(18043) → AgentCard

## STRIDE
| ID | Severity | Threat | Mitigation |
|----|----------|--------|------------|
| T-41-01 | MED | 多 Agent 循环失控 | LoopAgent 用 CountLoopStrategy(n) 有限次；ModelCallLimitHook |
| T-42-01 | MED | Supervisor 误调子 Agent | 子 Agent .description() 明确职责；AgentTool.create 白名单工具集 |
| T-43-01 | HIGH | Nacos 默认口令入库/生产误用 | README 安全提示（对齐 34 T-34-01）；凭证仅本地开发 yml，注明生产须更换 |
| T-43-02 | MED | 硬编码远程 Agent URL 绕过发现 | 使用 NacosAgentCardProvider + discovery.enabled，禁止写死对端 URL |
</threat_model>
